A complete end-to-end Business Intelligence project developed in Power BI for analyzing the operational and financial performance of an international shipping company.
The project covers the entire BI workflow, including data modeling, Power Query transformations, DAX calculations, interactive dashboards, forecasting, scenario analysis, and executive reporting.

## 🛠 Power BI Features Used

### Data Modeling
* Star Schema
* One-to-Many Relationships
* Calendar Table
* Data Validation

### Power Query
* Data Cleaning & Transformation
* Data Type Conversion
* Calculated Columns

### DAX Measures

The project includes more than 40 business measures, including:
* Total Revenue, Profit, Shipment
* Average Freight Rate, Revenue per Shipment, Average Delay
* Top Customer Revenue, Top Route Revenue
* Projected Profit, Profit Difference %
